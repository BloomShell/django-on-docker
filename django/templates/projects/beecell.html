Automated Stocks Valuation System

Automated Valuations are performed using structured financial data, coming from the SEC-Edgar RESTful API, hich are stored in a relational database (PostgreSQL-DB). Financial data includes +7000 companies across the world (United States, Canada, United Kingdom, Bermuda, Singapore, Ireland, China, Hong Kong, etc.). The Valuation System uses common Stock Pricing methods such as:
  • Multiples
  • Valuations (Discounted Cash Flow, Peter Lynch Fair Value, Projected Cash Flows, etc.)
  • Factor Metrics (Financial Strength, Profitability, Momentum, Growth, etc.)
The Orchestrator (Neural Network) produces an expected return, for each company, based on an attention mechanism that weights each input based on each observation trait (industry, size, location). Final adjustments are performed using consensus data relative to future expectations.

Black-Litterman

Valuations are used in the creation of a diversified portfolio (according to factors, industries, and geographic areas), active for the fiscal quarter, created through the Black-Litterman optimization through the expected returns generated by the Automatic Valuation System.
